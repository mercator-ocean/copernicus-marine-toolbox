Release 2.2.4
==============

Patch release with minor improvements and bug fixes.

General
--------

* Added explicit support for Python 3.14.
* Added constraint on ``click`` versions. See documentation :ref:`about click <click-issue>`.
* Added documentation about issues with ``netcdf4==1.7.3`` on Linux systems. See documentation :ref:`about netcdf4 <netcdf4-issue>`.

Subset
------

New Features
^^^^^^^^^^^^

* Released the constraint on ``--minimum-depth`` and ``--maximum-depth`` arguments in the ``subset`` command to allow negative values in the Command Line Interface. Negative values were already supported in the Python API.

Fixes
^^^^^

* Fixed an issue where the Toolbox wouldn't work with Dask distributed. The Toolbox is now compatible with Dask distributed setups.

Get
----

Fixes
^^^^^

* Fixed an issue where we would list more files than necessary if a dataset ID started with the same characters as another dataset ID when using the ``get`` command.
